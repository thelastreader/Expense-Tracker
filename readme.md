🎯 Expense Manager
A simple, interactive command-line expense tracking application built in Python.

✨ Features
➕ Add/Update Expenses - Easily add new expenses or update existing ones
📊 View Expenses - Display all your tracked expenses
🗑️ Delete Expenses - Remove unwanted expense entries
🚪 User-friendly Menu - Clean, emoji-enhanced interface
📋 Prerequisites
Python 3.6+
No external dependencies required
🚀 Quick Start
Clone or Download the project

Run the application:

bash

Copy code
python main.py
Enjoy tracking your expenses! 🎉

Reports: Create summary reports in view_expense.py
